# ReLogic_KontakAutomatorBundle

## Apa itu bundle ini?
Bundle ini adalah kumpulan alat (tools) sederhana untuk membantu Anda mengambil (menyimpan), mengelola, dan menyinkronkan kontak dari aplikasi Telegram ke format yang mudah digunakan seperti Excel, CSV, dan VCF (vCard). Tidak perlu pengetahuan pemrograman lanjutan untuk menjalankan alat ini.

## Fitur Utama
- Mengambil (mengekspor) semua kontak Telegram Anda ke file Excel, CSV, dan VCF.
- Menyiapkan kontak agar bisa diimpor ke perangkat lain atau layanan lain.
- Menyinkronkan kontak Telegram dengan file Excel Anda.

## Cara Instalasi & Menjalankan
1. **Siapkan Python**
   - Pastikan komputer Anda sudah terpasang Python 3.8 atau lebih baru. Download di https://python.org jika belum ada.

2. **Download & Ekstrak Bundle**
   - Ekstrak semua file bundle ini ke satu folder di komputer Anda.

3. **Install Kebutuhan Otomatis**
   - Buka Command Prompt (Windows) atau Terminal (Mac/Linux) di folder bundle ini.
   - Jalankan perintah berikut:
     ```bash
     pip install -r requirements.txt
     ```

4. **Siapkan API Telegram**
   - Kunjungi https://my.telegram.org, login, lalu buat API_ID dan API_HASH (ikuti petunjuk di website Telegram).
   - Simpan API_ID dan API_HASH Anda.

5. **Jalankan Scraper Kontak Telegram**
   - Jalankan perintah berikut:
     ```bash
     python relogic_tg_scraper/main.py
     ```
   - Ikuti instruksi login Telegram di layar. Kontak Anda akan tersimpan otomatis di folder `example_output/`.

6. **Jalankan Tool Lain (Opsional)**
   - Untuk menyiapkan kontak ke perangkat lain:
     ```bash
     python relogic_contact_injector/main.py
     ```
   - Untuk menyinkronkan kontak ke Excel:
     ```bash
     python relogic_tg_excel_sync/main.py
     ```

## Cara Otomatis: Jalankan Semua Sekaligus
Jika Anda ingin menjalankan semua proses (scrape, inject, sync Excel) sekaligus tanpa repot, cukup:

1. Cari file bernama `run_all.bat` di folder utama bundle ini.
2. Klik dua kali (double click) file tersebut.
3. Ikuti instruksi di layar, semua proses akan berjalan otomatis satu per satu.

Hasilnya tetap akan muncul di folder `example_output/`.

## Penjelasan Istilah
- **CSV**: File tabel sederhana, bisa dibuka di Excel.
- **VCF/vCard**: Format standar untuk menyimpan kontak, bisa diimpor ke HP atau Gmail.
- **Excel/XLSX**: File spreadsheet Microsoft Excel.
- **API_ID & API_HASH**: Kode unik dari Telegram agar aplikasi bisa mengakses akun Anda (aman, tidak membocorkan password).

# Struktur Folder
- `relogic_tg_scraper/` : Ekspor kontak Telegram ke CSV & VCF
- `relogic_contact_injector/` : Parse & injeksi kontak ke device/layanan lain
- `relogic_tg_excel_sync/` : Sinkronisasi kontak Telegram dengan file Excel

# Cara Pakai
1. **Scrape Kontak Telegram**
   - Isi `API_ID` dan `API_HASH` di environment variable atau di kode.
   - Jalankan:
     ```bash
     python relogic_tg_scraper/main.py
     ```
   - Output: `example_output/contacts.csv` dan `example_output/sample.vcf`

2. **Inject Kontak**
   - Jalankan:
     ```bash
     python relogic_contact_injector/main.py
     ```
   - Output: Siap di-inject ke device/layanan lain (implementasi placeholder)

3. **Sync ke Excel**
   - Jalankan:
     ```bash
     python relogic_tg_excel_sync/main.py
     ```
   - Output: `example_output/contacts_synced.xlsx`

## Troubleshooting / Mengatasi Error Umum

**1. Error: No module named 'telethon'**
- Solusi: Jalankan `pip install telethon` di terminal.

**2. Error: No module named 'relogic_tg_scraper' (atau package lain)**
- Solusi: Jalankan script dengan perintah `python -m relogic_tg_scraper.main` (bukan `python relogic_tg_scraper/main.py`).
- Pastikan sudah ada file `__init__.py` di setiap folder utama.

**3. Error: FileNotFoundError: example_output/...**
- Solusi: Pastikan folder `example_output` sudah ada, atau jalankan ulang script setelah update (sekarang folder akan dibuat otomatis).

**4. Error: AttributeError: 'TelegramClient' object has no attribute 'get_contacts'**
- Solusi: Script sudah diperbaiki, pastikan Anda menggunakan versi terbaru dari bundle ini.

**5. Tips Umum Menjalankan Script**
- Selalu jalankan script dari folder utama proyek dengan perintah `python -m nama_package.nama_script`.
- Jika ada error lain, baca pesan error di terminal dan cek README.md bagian ini.
